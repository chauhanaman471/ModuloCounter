This file contains the declaration of the ModuloNCounter class.

The ModuloNCounter class implements a multi-digit counter, where each digit is an instance of the ModuloNDigit class. It support different counter base (Binary, Octal, Decimal, Hexadecimal) and handles overflow between digits.
